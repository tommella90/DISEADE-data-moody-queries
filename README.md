# MOODY DATA QUERIES

La repo contiene 3 cartelle: 
- **data_raw**: ogni cartella dentro **Data Raw** dovrebbe corrispondere a una cartella di Moody (eg: *ownership history*)
- **queries**: ogni notebook in **queries** dovrebbe dare come output un file (stata, parquet...) con dati processati
- **data_processed**: contiene i dati processati dalle queries
